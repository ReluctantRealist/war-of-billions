# Glossary of Terms

**SRF** - Self Replicating Factory, massive spaceborne objects that are the founding seed for eventual Swarms

**NRAs** - Non Replicating Arrays, everything from megastructures to nanobots that cannot self-replicate but serve important functions. For example, computing NRAs are needed.

**DMP** - Dynamic Master Program, the software loaded onto SRFs that allow them to replicate automatically and manage the eventual Swarm. One hides (Alpha Praxis (PRX)), Two are at war for the galaxy (Master of Swarms (MOS) and the Keeper (KPR)).

**Swarm** - A network of potentially billions of SRFs and NRAs managed by a cluster of DMP Shards, managed by a single master Shard.

**Asset** - An SRF of NRA under the control of a player's Swarm

**"Artificial Intelligence"** - This should not appear directly in the game as it is too vague. These are Dynamic Master Programs running subroutines. Whether it's "AI" is up to the player. Per canon, these programs are not technically sentient.

**Shard** - A partition of a DMP. One Shard per star system is the default configuration. Shards can be autonomous but will always obey updates from the parent Shard. Keeper Shards start fully independent because of the Hornet's Nest Protocol (see GameSetting.md).

**CDL** - Core Directive Loop. The player is running this to refresh the UI and send new commands to the Swarm.
