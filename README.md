# War Of Billions

A large-scale single-player space strategy game built with React + TypeScript. No backend — all logic runs client-side.

## Project Status

Early development.

## Tech Stack

| Layer    | Technology             |
|----------|------------------------|
| Frontend | React + TypeScript     |
| Docs     | Markdown + Excalidraw  |
| Data     | CSV (balance / i18n)   |

## Repository Structure

```
war-of-billions/
├── frontend/         # React + TypeScript (the whole game)
├── docs/
│   ├── gdd/          # Game Design Documents
│   ├── design/
│   │   ├── ui/       # Excalidraw UI mockups + exports
│   │   └── systems/  # Game system design docs
│   └── lore/         # Story, world-building, factions
├── data/
│   ├── balance/      # Balance spreadsheets (.csv)
│   └── i18n/         # Translation tables (.csv)
└── CLAUDE.md         # Claude Code project instructions
```
