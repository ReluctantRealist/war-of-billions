# War Of Billions

A large-scale strategy/action game built with Java (backend) and React + TypeScript (frontend).

## Project Status

Early development.

## Tech Stack

| Layer    | Technology             |
|----------|------------------------|
| Backend  | Java (Spring Boot)     |
| Frontend | React + TypeScript     |
| Docs     | Markdown + Excalidraw  |
| Data     | CSV (balance / i18n)   |

## Repository Structure

```
war-of-billions/
├── backend/          # Java Spring Boot server
├── frontend/         # React + TypeScript client
├── docs/
│   ├── gdd/          # Game Design Documents
│   ├── design/
│   │   ├── ui/       # Excalidraw UI mockups + exports
│   │   └── systems/  # Game system design docs
│   └── lore/         # Story, world-building, factions
├── data/
│   ├── balance/      # Balance spreadsheets (.csv)
│   └── i18n/         # Translation tables (.csv)
└── CLAUDE.md         # Claude Code project instructions
```
